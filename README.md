# Enterprise-Big-Data-Engineering-labs
Collection of hands-on processing engines, data frameworks, and database components that sit on top of or integrate with the Hadoop ecosystem with a focus on distributed data processing and big data engineering practices.

The objective of this portfolio is to demonstrate practical experience in distributed data processing, data storage, and large-scale data management technologies commonly used in enterprise Big Data environments.

The repository is organized by technology, allowing each topic to be explored independently while providing a structured record of learning progress and technical implementation.

---

## Technologies Covered

### Hadoop MapReduce

* Distributed batch processing
* Mapper and Reducer development
* Data aggregation and transformation
* Custom MapReduce workflows
* Hadoop job execution and monitoring

### Apache Hive

* Data warehousing concepts
* Hive Query Language (HQL)
* Data definition and manipulation
* Partitioning and table management
* Analytical query development

### Apache HBase

* NoSQL database operations
* Column-family data modeling
* Table creation and administration
* Data insertion and retrieval
* HBase shell operations

## Repository Structure

```text
MapReduce/
Hive/
HBase/
Notes/
Future-Technologies/
```

Each laboratory directory typically contains:

```text
Lab-XX/
├── assignment.pdf
├── source_code/
├── screenshots/
└── report.pdf
```

### assignment.pdf

Contains the original assignment requirements and objectives.

### source_code

Contains implementation files, scripts, queries, or programs developed to complete the assignment.

### screenshots

Contains execution results, outputs, Hadoop UI captures, Hive queries, HBase operations, and validation evidence.

### report.pdf

Optional documentation describing implementation details, observations, and conclusions.
